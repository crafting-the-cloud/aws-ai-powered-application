# Internal Corporate Training Catalog

## Cloud & DevOps Track

### Course ID: CLD-101
**Course Name:** AWS for Enterprise Architects
**Description:** A deep dive into AWS core services with a focus on security, scalability, and cost optimization for large-scale enterprise projects. Covers advanced networking, IAM policies, and infrastructure as code with Terraform.
**Target Audience:** Senior Developers, DevOps Engineers, Cloud Engineers.
**Skills Covered:** AWS Solutions Architecture, Terraform, VPC, IAM, Cost Management.

### Course ID: CLD-201
**Course Name:** Mastering Kubernetes on EKS
**Description:** From zero to hero with Amazon EKS. This course covers Docker fundamentals, Kubernetes architecture, deploying microservices, and managing production clusters with monitoring and logging tools like Prometheus and Grafana.
**Target Audience:** DevOps Engineers, Backend Developers.
**Skills Covered:** Kubernetes, Docker, EKS, Microservices, Prometheus.

## AI/ML Track

### Course ID: AIML-101
**Course Name:** Applied Machine Learning with SageMaker
**Description:** A practical, hands-on course teaching how to build, train, and deploy machine learning models using Amazon SageMaker. Covers common algorithms, data preprocessing, and MLOps principles.
**Target Audience:** Software Developers, Data Analysts.
**Skills Covered:** Python, Scikit-learn, Pandas, Amazon SageMaker, MLOps.

### Course ID: AIML-301
**Course Name:** Advanced GenAI: Fine-Tuning Models with Bedrock
**Description:** An expert-level workshop on customizing foundation models available in Amazon Bedrock. Learn how to prepare a dataset for fine-tuning, run a tuning job, and evaluate model performance for specific company use cases.
**Target Audience:** AI/ML Engineers, Senior Cloud Architects.
**Skills Covered:** Generative AI, Amazon Bedrock, Model Fine-Tuning, Prompt Engineering.

## Software Development Track

### Course ID: SDE-210
**Course Name:** Serverless Development Patterns
**Description:** Learn how to build highly scalable and cost-effective applications using serverless patterns on AWS. Focuses on Lambda, API Gateway, DynamoDB, and EventBridge. Includes hands-on labs for building a real-world serverless application.
**Target Audience:** All software developers.
**Skills Covered:** AWS Lambda, API Gateway, DynamoDB, Serverless Framework.
